# Quiz Naruto

## Descrição
O Quiz Naruto é uma aplicação web interativa que permite ao usuário descobrir qual personagem do universo Naruto mais combina com sua personalidade.

O sistema apresenta 10 perguntas e, com base nas respostas escolhidas, calcula um resultado final entre três personagens principais:
- Naruto Uzumaki  
- Sasuke Uchiha  
- Sakura Haruno  



## Objetivo
Aplicar conceitos de desenvolvimento web front-end, incluindo:
- Manipulação do DOM  
- Programação Orientada a Objetos (POO) em JavaScript  
- Estruturação de interfaces com HTML e CSS  
- Criação de aplicações interativas  



## Funcionalidades
- Tela inicial com introdução ao quiz  
- Botão de início da jornada  
- Exibição dinâmica de perguntas  
- Sistema de pontuação por resposta  
- Avanço automático entre perguntas  
- Cálculo do resultado final  
- Exibição do personagem com imagem e pontuação  
- Botão para refazer o quiz  
- Interface estilizada com animações e efeitos visuais  



## Como Funciona
As perguntas são armazenadas em um array no JavaScript. Cada resposta possui pontuações específicas para cada personagem.

Ao selecionar uma alternativa, os pontos são acumulados e a próxima pergunta é exibida automaticamente. Ao final do quiz, o personagem com maior pontuação é apresentado como resultado.



## Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript (ES6+)  



## Como Executar o Projeto
1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` em um navegador  
3. Clique em "Iniciar Jornada"  
4. Responda às perguntas  
5. Veja o resultado final
